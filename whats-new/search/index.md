title: Search
date: April 4, 2026
---

Full-text search across all pages, records, and commands with **Cmd+K** / **Ctrl+K**. Filter by type, field, property value, or repository using the advanced syntax — `type:record property:status=approved repo:my-repo`.

Search is powered by a GitHub Actions workflow that indexes your repository on every push to main and stores the index directly in your repo. No content leaves your GitHub organization.
