title: Slash Commands
---

Inside any document, type `/` to open the block insertion menu. Available blocks:

- `/heading` ‚Äî H1, H2, or H3 heading
- `/bullet` ‚Äî bulleted list
- `/numbered` ‚Äî numbered list
- `/table of contents` ‚Äî auto-generated TOC from headings in the page
- `/columns` ‚Äî multi-column layout block
- `/code` ‚Äî code block with syntax highlighting
- `/divider` ‚Äî horizontal rule
