Type `/` inside any document to open the block insertion menu.

## Text & headings

- `/Heading 1` ‚Äî H1 heading
- `/Heading 2` ‚Äî H2 heading
- `/Heading 3` ‚Äî H3 heading
- `/Quote` ‚Äî blockquote
- `/Divider` ‚Äî horizontal rule

## Lists

- `/Bullet List` ‚Äî unordered list
- `/Numbered List` ‚Äî ordered list
- `/To-do List` ‚Äî checkbox task list
- `/Toggle List` ‚Äî collapsible toggle block

## Toggles

- `/Toggle H1` ‚Äî H1-sized collapsible toggle
- `/Toggle H2` ‚Äî H2-sized collapsible toggle
- `/Toggle H3` ‚Äî H3-sized collapsible toggle

## Layout

- `/2 Columns` ‚Äî two-column layout
- `/3 Columns` ‚Äî three-column layout
- `/4 Columns` ‚Äî four-column layout
- `/Table` ‚Äî insert a 3√ó3 table
- `/Table of Contents` ‚Äî auto-generated TOC from page headings

## Media & embeds

- `/Image` ‚Äî embed an image by URL
- `/Code Block` ‚Äî code block with syntax highlighting
- `/Equation` ‚Äî KaTeX / LaTeX math block
- `/Callout` ‚Äî highlighted callout block
- `/Emoji` ‚Äî emoji picker

## Pages

- `/Page` ‚Äî create and link a new sub-page
- `/Link to page` ‚Äî link to an existing page
