---
title: Commands
---

Lightworks exposes a command palette and a set of keyboard shortcuts for navigating pages, running actions, and managing your workspace.

## Command palette

Open the command palette with **Cmd+K** (Mac) or **Ctrl+K** (Windows/Linux). From here you can search pages, run workspace actions, and navigate anywhere in the app.

## Keyboard shortcuts

Press **Cmd+/** (Mac) or **Ctrl+/** (Windows/Linux) to open the keyboard shortcuts reference. Shortcuts are grouped by scope:

| Scope | Action | Keys |
|---|---|---|
| Global | Open command palette | Cmd+K |
| Global | Keyboard shortcuts | Cmd+/ |
| Document | Save changes | Cmd+S |
| Document | Toggle frontmatter | Cmd+Shift+F |
| Block | New line | Enter |
| Block | Hard break | Shift+Enter |

## Slash commands

Inside any document, type `/` to open the block insertion menu. Available blocks:

- `/heading` — H1, H2, or H3 heading
- `/bullet` — bulleted list
- `/numbered` — numbered list
- `/table of contents` — auto-generated TOC from headings in the page
- `/columns` — multi-column layout block
- `/code` — code block with syntax highlighting
- `/divider` — horizontal rule
