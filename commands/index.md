Type `/` inside any document to open the block insertion menu.

## Text & headings

- `/Heading 1` — H1 heading
- `/Heading 2` — H2 heading
- `/Heading 3` — H3 heading
- `/Quote` — blockquote
- `/Divider` — horizontal rule

## Lists

- `/Bullet List` — unordered list
- `/Numbered List` — ordered list
- `/To-do List` — checkbox task list
- `/Toggle List` — collapsible toggle block

## Toggles

- `/Toggle H1` — H1-sized collapsible toggle
- `/Toggle H2` — H2-sized collapsible toggle
- `/Toggle H3` — H3-sized collapsible toggle

## Layout

- `/2 Columns` — two-column layout
- `/3 Columns` — three-column layout
- `/4 Columns` — four-column layout
- `/Table` — insert a 3×3 table
- `/Table of Contents` — auto-generated TOC from page headings

## Media & embeds

- `/Code Block` — code block with syntax highlighting
- `/Equation` — KaTeX / LaTeX math block
- `/Callout` — highlighted callout block
- `/Emoji` — emoji picker

## Pages

- `/Page` — create and link a new sub-page
- `/Link to page` — link to an existing page

## Connections

Available once the corresponding provider is connected in **Settings → Connections**. See [Connections](/docs/connections).

- `/Linear ticket` — insert an inline reference to a Linear issue
- `/Jira issue` — insert an inline reference to a Jira issue
